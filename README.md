## 💻 실행 방법

1. **원격 저장소 복제**

```bash
$ git clone https://github.com/JeongHwan-dev/twitter-clone-coding.git
```

2. **프로젝트 폴더로 이동**

```bash
$ cd twitter-clone-coding
```

3. **필요한 node_modules 설치**

```bash
$ npm install
```

4. **리액트 앱 실행**

```bash
$ npm start
```
